# HDRO-RShiny-Dashboard

## Overview

This project is an interactive **R Shiny** application built using data from the Human Development Report Office of the United Nations Development Programme (UNDP).

The app allows users to explore human development indicators through:

- A dynamic data table for easy viewing and filtering  
- Interactive visualisations that highlight key trends and relationships within the dataset  

It provides a simple interface to analyse HDRO data and gain insights through both tabular and graphical representations.
